## 数据库连接

```bash
# 进入 psql
docker exec sgz-postgres psql -U sgz -d sgz

# 导出 CSV
docker exec sgz-postgres psql -U sgz -d sgz -c "\COPY (...) TO '/tmp/xxx.csv' WITH CSV HEADER ENCODING 'UTF8';"
```

> **注意**：导出 CSV 后需加 UTF-8 BOM 才能在 Excel 中正确显示中文：
> ```bash
> printf '\xEF\xBB\xBF' | cat - /tmp/xxx.csv > ~/Desktop/xxx.csv
> ```
