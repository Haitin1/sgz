## skills 表结构

```sql
CREATE TABLE skills (
  id           SERIAL PRIMARY KEY,
  name         VARCHAR(50) NOT NULL UNIQUE,
  category     VARCHAR(10),          -- 主动/指挥/被动/内政/兵种/突击/阵法
  troop_type   VARCHAR(10),
  trigger_prob NUMERIC(5,2),
  target       VARCHAR(20),
  duration     VARCHAR(20),
  description  TEXT,
  effect_json  JSONB,
  is_inherited BOOLEAN DEFAULT FALSE,
  is_event     BOOLEAN DEFAULT FALSE,
  is_builtin   BOOLEAN DEFAULT FALSE,
  quality      VARCHAR(5),           -- S/A/B 等
  source       VARCHAR(30),
  conflict     TEXT,
  effect       VARCHAR(30),
  created_at   TIMESTAMP DEFAULT NOW()
);
```

## advisor_skills 表结构

```sql
CREATE TABLE advisor_skills (
  id             SERIAL PRIMARY KEY,
  general_name   VARCHAR(30) NOT NULL,
  skill_name     VARCHAR(50) NOT NULL UNIQUE,
  attribute      VARCHAR(5),          -- 风/林/火/山
  troop_restrict VARCHAR(20),         -- 兵种限制，NULL 表示无限制
  cost           INTEGER,             -- 计略点消耗
  target         VARCHAR(50),
  is_passive     BOOLEAN DEFAULT FALSE,
  daily_limit    INTEGER,             -- 每日使用次数，NULL 表示无限
  is_implemented BOOLEAN DEFAULT TRUE, -- FALSE = 未实装/预定追加
  description    TEXT,
  effect_json    JSONB,               -- 结构化数值，含 base/max
  created_at     TIMESTAMP DEFAULT NOW()
);
```

### effect_json 格式约定

所有数值字段统一用 `{"base": X, "max": Y}` 表示初始值和满熟练度值，例如：

```json
{
  "speed_bonus": {"base": 20, "max": 40},
  "duration_hour": 2
}
```
