## generals 表

武将核心表，**属性存的是 Lv1 基础值**，前端按等级动态计算：`显示值 = 基础值 + 成长 × (Lv - 1)`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | varchar(50) | 武将名 |
| faction | varchar(10) | 势力（魏/蜀/吴/群/汉） |
| role | varchar(20) | 定位（统帅/谋士/猛将等） |
| command_value | smallint | 统御值（1-7） |
| cavalry/shield/bow/spear/machine/navy | varchar(2) | 各兵种适性等级（S/A/B/C） |
| innate_skill | varchar(50) | 自带战法名 |
| inherited_skill | varchar(50) | 可传承战法名 |
| strength/intelligence/leadership/speed/politics/charisma | numeric(8,2) | 六维基础值（Lv1） |
| str/int/lead/spd/pol/cha\_growth | numeric(6,2) | 六维成长值 |
| max\_strength ... max\_total | smallint | 各维满级上限 |
| four_stats_sum | smallint | 武智统速四维之和 |
| is_collector | boolean | 是否有典藏版 |
| is_dynamic | boolean | 是否有动态立绘 |
| is_strategist | boolean | 是否可担任军师 |
| strategist_skill | varchar(50) | 军师技名称 |
| strategy_points | smallint | 军师技计略点 |
| exclusive_title | varchar(50) | 专属称号 |

---

## skills 表

战法数据，前端战法选择面板从这里读取。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | varchar(50) | 战法名 |
| category | varchar(10) | 分类：主动/指挥/被动/内政/兵种/突击/阵法 |
| quality | varchar(5) | 品质：S/A/B |
| trigger_prob | numeric(5,2) | 触发概率（0~1） |
| target | varchar(20) | 目标：自身/敌军单体/敌军全体等 |
| troop_type | varchar(10) | 兵种限制（NULL=无限制） |
| description | text | 完整效果描述 |
| effect | varchar(30) | 简短效果标签 |
| effect_json | jsonb | 结构化数值 |
| is_inherited | boolean | 是否可传承 |
| is_event | boolean | 是否活动/限定战法 |
| is_builtin | boolean | 是否自带战法 |
| source | varchar(30) | 来源 |
| conflict | text | 互斥战法 |

---

## advisor_skills 表

军师技数据，按武将归属，分风/林/火/山四属性。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| general_name | varchar(30) | 所属武将名 |
| skill_name | varchar(50) | 军师技名称 |
| attribute | varchar(5) | 属性：风/林/火/山 |
| troop_restrict | varchar(20) | 兵种限制（NULL=无限制） |
| cost | int | 计略点消耗 |
| target | varchar(50) | 目标范围 |
| is_passive | boolean | 是否被动触发 |
| daily_limit | int | 每日使用次数（NULL=无限） |
| is_implemented | boolean | 是否已实装（FALSE=预定追加） |
| description | text | 效果描述 |
| effect_json | jsonb | 结构化数值，格式 `{"base": X, "max": Y}` |

---

## affinities 表

缘分组合数据，武将凑齐对应组合触发加成。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | varchar(30) | 缘分名称 |
| min_members | smallint | 最少需要几人（默认3） |
| members | text | 成员列表（逗号分隔武将名） |
| effect | text | 效果描述 |

---

## equipment_skills 表

装备（宝物/器械）的技能数据。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | varchar(50) | 技能名 |
| eq_type | varchar(20) | 装备类型（宝物/攻城器械等） |
| source | varchar(30) | 来源 |
| strength_level | varchar(20) | 强化等级 |
| exclusive_for | varchar(50) | 专属武将（NULL=通用） |
| description | text | 效果描述 |
| effect_json | jsonb | 结构化数值 |

---

## military_books 表

兵书数据，按体系和类型分类。

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | varchar(20) | 兵书名 |
| book_system | varchar(10) | 体系（孙子/吴子等） |
| book_type | varchar(10) | 类型（攻击/防御/速度等） |
| description | text | 效果描述 |
