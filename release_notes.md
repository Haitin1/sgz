# Release Notes

---

## 2026-04-23

### 队伍兵种锁定
- 我方/敌方队伍头部新增兵种下拉框
- 选定后队内3名武将兵种统一锁定，其余标签变灰不可切换
- 适应等级仍可单独修改，不受锁定影响

### 兵种适性标签交互优化
- 点击整个标签触发适应等级选择（原来只能点字母）
- 使用透明全覆盖 select 实现，操作更方便

### 兵种适性标签 UI
- 替换原来的兵种/适应下拉框，改为彩色适性标签
- 5个兵种标签颜色对应等级（金=S、绿=A、蓝=B、灰=C）
- 点击标签选中兵种，等级可临时修改（不写入数据库）
- 适性数据从数据库武将表读取并自动填充

### 敌方队伍（B）结构修复
- 补充政治/魅力属性行（之前缺失）
- 修正属性输入框的 syncBase/syncGrowth 调用（之前只设 dirty=true，导致保存逻辑错误）

### 默认武将从数据库自动填充
- init() 启动时自动从 generalsData 匹配6名默认武将
- 成长值、政治、魅力全部从数据库读取，不再显示为0

### 代码仓库建立
- 初始化 git 仓库，代码与文档合并到 https://github.com/Haitin1/sgz
- 文档移至 docs/ 目录，.gitignore 排除 __pycache__ 和密钥文件
